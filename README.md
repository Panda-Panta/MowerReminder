# MowerReminder

**通过读取Mower的运行日志，在任务开始前对用户进行提醒**

## 功能支持

### 核心提醒
- 倒计时显示
- 剩余时间低于设定阈值时窗口背景闪烁提醒
- 自定义倒计时闪烁阈值（秒）

### 数据源模式
- **本地模式**：读取指定日志文件，周期性扫描解析任务时间
- **网址模式**：通过 WebSocket 实时接收日志，动态解析任务时间
- 两种模式可在设置界面切换

### 通知提醒
- 支持 3 个独立的倒计时通知（可分别设置分钟数并启用/禁用）
- 任务结束/休息开始通知（检测到新任务时提醒休息剩余分钟数）
- Windows 原生气泡通知（Windows 10\11）

### 窗口与界面
- 窗口可拖拽移动
- 一键置顶/取消置顶
- 启动时最小化到系统托盘
- 深色主题背景

### 程序运行
- 单实例运行（防止重复启动，再次运行激活已有窗口）
- 配置文件自动保存和加载
